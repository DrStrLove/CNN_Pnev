# CNN_Pnev

Исследовательский прототип системы, которая анализирует медицинские изображения и классифицирует их с помощью сверточных нейронных сетей для выявления патологий.

## Описание

В медицинской диагностике важную роль играет анализ изображений, таких как рентгеновские снимки. Ручная интерпретация требует высокой квалификации и может быть подвержена ошибкам.
Цель проекта: разработать модель, способную автоматически классифицировать изображения (например, выявлять признаки пневмонии) на основе обучающей выборки.
Система ориентирована на применение в задачах компьютерного зрения в медицине и автоматизации первичной диагностики.

## Подход

Используется подход на основе глубокого обучения:
- Сверточная нейронная сеть (CNN)
  Модель извлекает пространственные признаки изображений и обучается различать классы на основе визуальных паттернов.
- Обучение модели
  Сеть обучается на размеченных данных с использованием функции потерь для задачи классификации.
- Предсказание
  Для нового изображения модель возвращает вероятность принадлежности к каждому классу.

## Данные и признаки

Система работает с изображениями:
- входные данные: медицинские изображения (например, рентген грудной клетки)
- изображения приводятся к единому размеру и нормализуются
- возможно применение аугментаций (повороты, отражения, масштабирование) для увеличения обучающей выборки
